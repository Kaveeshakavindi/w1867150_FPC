## Getting started

Step 1 : Activate virtual environment

```bash
source .venv/bin/activate
```

Step 2 : Run backend

```bash
cd w1867150_IPD/Backend
python -m uvicorn api:app --reload
```

Step 3 : Run frontned

required npm version : 20

```bash
cd w1867150_IPD/Frontend/synapse-frontend
npm run dev
```
